4.3 Errors 31-40

E31: ERROR_FUNCTION_NOT_FOUND

If you try to call a function that does not exist, this error will be thrown.

E32: ERROR_FUNCTION_ARG_NOT_CORRECT_AMOUNT

If you call a function with the wrong amount of arguments, this error will be thrown.

E33: ERROR_INVALID_LITERAL

If you try to use a literal that is not valid, this error will be thrown.

E34: ERROR_INVALID_OPERATOR

If you try to use an operator that is not valid, this error will be thrown.

E35: ERROR_CANT_USE_TYPE_IN_ADDITION

If you try to add with something you can't add with, this error will be thrown.
(currently theres no type that can't be used in addition, so this error will never be thrown)

E36: ERROR_CANT_CONVERT_TO_STRING

If something can not be converted into a string, this error will be thrown.
In practice everything can be converted into a string, so this error will never be thrown.

E37: ERROR_FLOATING_CANT_BE_UNSIGNED

Unused

E38: ERROR_TYPE_NOT_SUBTRACTABLE

E39: ERROR_CANT_USE_TYPE_IN_MULTIPLICATION

E40: ERROR_CANT_USE_TYPE_IN_DIVISION

Because Dosato automatically converts types, these errors will never be thrown.
