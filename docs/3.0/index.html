3.0 Standard library

Dosato by default has a selection of build in functions and variables, to help you build efficient code.

In this chapter, we will list and explore all the functions and variables Dosato has to offer!

General Constants

TRUE

TRUE is a constant defined in the standard library, it is the value of TRUE.
It's value internally is TRUE

FALSE

FALSE is a constant defined in the standard library, it is the value of FALSE.
It's value internally is FALSE

MAXINT

MAXINT is a constant defined in the standard library, it is the maximum value of an integer.
It's value internally is 2147483647

MININT

MININT is a constant defined in the standard library, it is the minimum value of an integer.
It's value internally is -2147483648

MAXLONG

MAXLONG is a constant defined in the standard library, it is the maximum value of a long.
It's value internally is 9223372036854775807

MINLONG

MINLONG is a constant defined in the standard library, it is the minimum value of a long.
It's value internally is -9223372036854775808

General functions

VOID BREAK (LONG break_amount = 1)

BREAK is a function defined in the standard library, it is used to break out of a loop.
BREAK takes max 1 argument, the number of loops to break out of.

VOID CONTINUE ()

CONTINUE is a function defined in the standard library, it is used to skip the current iteration of a loop.
CONTINUE takes no arguments.

ANY RETURN (ANY value = ?)

RETURN is a function defined in the standard library, it is used to return a value from a function.
RETURN takes max 1 argument, the value to return.
