2.4 The _ variable

The _ variable is a special variable that is used to store the result of the last function call.
It's also used to store the error code when an error occurs.
The _ variable is read-only and cannot be modified.
The _ variable is a global variable and can be accessed from anywhere in the program.

Example

In this example, the _ variable is used to store the result of the last function call: 

MAKE FUNC INT add(INT a, INT b) {
    DO RETURN (a + b);
};

DO add(5, 10);
DO SAY (_); // 15

Function calls in expressions are automatically seen as _ variables. 

DO SAY (add(5, 10)); // 15, the function call is converted to _, so the interpreter sees it as DO SAY (_);

The _ variable is also used to store the error message when an error occurs.
It turns into an INT when an error occurs.
Example: 

DO {
    DO SAY (notDefined); // Error is thrown
} CATCH {
    DO SAY (_); // The error message is stored in the _ variable
};
