4.6 Errors 61-70

E61: ERROR_VARIABLE_ALREADY_EXISTS

This error occurs when you try to create a variable that already exists.
You can only create a variable once.
If you want to change the value of a variable, use the SET keyword.

E62: ERROR_WHILE_NOT_LAST

This error occurs when you use the WHILE keyword in the middle of a DO block.
The WHILE keyword can only be used at the start of a DO block.
Example: 

DO {
    DO SAYLN ("Hello");
} WHILE (TRUE) WHEN (TRUE); // Error, WHILE must be at the end.

E63: ERROR_EXPECTED_ELSE

This error occurs when you use the WHEN keyword without an ELSE keyword.
When WHEN is at the end of a DO block, it's fine.
But when WHEN is in the middle of a DO block, it needs to be followed by an ELSE keyword.
Example: 

DO {
    DO SAYLN ("Hello");
} WHEN (TRUE) THEN SAYLN ("World"); // Error: Expected Else

E64: ERROR_EXTENSION_NOT_FINAL

This error occurs when you try to use an extension that is not final but wants to be final.
(CATCH|INTO) is an example of a final extension.
Example: 

DO {
    DO SAYLN ("Hello");
} INTO e THEN SAYLN ("World"); // Error: Extension not final

E65: ERROR_FUNCTION_ALREADY_EXISTS

This error occurs when you try to create a function that already exists.
You can only create a function with a certain name once.

E66: ERROR_INVALID_TYPE

When you use a type that doesn't exist, this error occurs.
Examples: (INT INT, VOID INT, etc), just any type combination that doesn't exist.

E67: ERROR_BREAK_OUTSIDE_OF_LOOP

E68: ERROR_CONTINUE_OUTSIDE_OF_LOOP

These errors occur when you use the BREAK or CONTINUE function outside of a WHILE or FOR loop.
BREAK and CONTINUE can only be used inside of loops.

E69: ERROR_TOO_FEW_ARGUMENTS

This error occurs when you call a system function with too few arguments.
The function you're calling requires more arguments than you're giving it.
Check chapter 3 for a list of standard library functions.

E70: ERROR_RETURN_OUTSIDE_OF_FUNCTION

This error occurs when you use the RETURN function outside of a user made function.
The RETURN function can only be used inside of a user made function.
