3.5 Time

Functions

INT TIME ()

Returns the current time in seconds.

STRING DATE (LONG timestamp = now)

Returns a string representation of the given timestamp.
The format is: DD-MM-YYYY

STRING DATETIME (LONG timestamp = now)

Returns a string representation of the given timestamp.
The format is: DD-MM-YYYY hh:mm:ss

LONG TIMESTAMP (LONG timestamp = now)

Returns a string representation of the given timestamp.
The format is: hh:mm:ss

LONG CLOCK ()

Returns the current time in milliseconds.

VOID SLEEP (DOUBLE seconds)

Waits excecution for the given amount of seconds (can be decimal).
