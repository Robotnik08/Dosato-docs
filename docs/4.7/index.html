4.7 Errors 71-80

E71: ERROR_MATH_DOMAIN_ERROR

This error occurs when you try to perform a mathematical operation that is not allowed.
For example, dividing by zero.
Or taking the square root of a negative number.
Examples: 

MAKE INT number = 5 / 0; // Error, division by zero
MAKE FLOAT number = ^/ -1; // Error, square root of negative number

E72: ERROR_FILE_NOT_FOUND

This error occurs when you try to access a file that does not exist.
For example, trying to read from a file that has not been created.

E73: ERROR_NUMBER_CANNOT_BE_NEGATIVE

This error occurs when you try to use a negative number where it is not allowed.
Occurs in certain system functions using arrays and strings.

E74: ERROR_START_GREATER_THAN_END

This error occurs when you try to use a start index that is greater than the end index.
Occurs in certain system functions using arrays and strings.

E75: ERROR_PERMISSION_DENIED

This error occurs when you try to access a file that you do not have permission to access.
For example, trying to read from a file that you do not have read permission for.

E76: ERROR_INVALID_EXTENSION

This error occurs when you try to use an extension that is not valid.
For example, not putting a THEN extension after an IF extension.

E77: ERROR_INCORRECT_RETURN_TYPE

Unused.

E78: ERROR_INTERNAL

E79: ERROR_UNKNOWN

E80: ERROR_AMOUNT

If any of these errors occur, please report them to the Developer.
These errors should never occur.
