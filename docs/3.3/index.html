3.3 Arrays

Functions

ARRAY ANY ARRAYSHIFT (ARRAY ANY arr)

Shifts the given array, removing the first element and returning a copy.

ARRAY ANY ARRAYREMOVE (ARRAY ANY arr, INT start, INT amount)

Removes the given amount of elements from the given array, starting at the given index, and returns a copy.

ARRAY ANY ARRAYINSERT (ARRAY ANY arr, INT start, ANY insert)

Inserts the given element into the given array, starting at the given index, and returns a copy.

ARRAY ANY ARRAYSLICE (ARRAY ANY arr, INT start, INT amount)

Returns a slice of the given array, starting at the given index and ending at the given index + amount.

INT ARRAYFINDINDEX (ARRAY ANY arr, ANY element)

Returns the index of the first occurence of the given element in the given array.
Returns -1 if the element is not found.

INT ARRAYLASTINDEX (ARRAY ANY arr, ANY element)

Returns the index of the last occurence of the given element in the given array.
Returns -1 if the element is not found.

BOOL ARRAYCONTAINS (ARRAY ANY arr, ANY element)

Returns true if the given element is found in the given array, false otherwise.

ARRAY ANY ARRAYREVERSE (ARRAY ANY arr)

Returns a copy of the given array, with the elements in reverse order.

ARRAY ANY ARRAYSORT (ARRAY ANY arr)

Returns a copy of the given array, with the elements sorted in ascending order.
Sorting is based of the numerical value of the elements.

ARRAY ANY ARRAYSORTFUNC (ARRAY ANY arr, STRING callback_name)

Returns a copy of the given array, with the elements sorted in ascending order.
Sorting is based of the return value of the given callback function.
Since Dosato can't pass functions as parameters, the name of the callback in string form must be passed instead.

ARRAY INT RANGE (INT end)
ARRAY INT RANGE (INT start, INT end, INT step = 1)

Returns an array of integers, starting at the given start index, ending at the given end index, and incrementing by the given step.
If no step is given, the default value of 1 is used.
If only 1 parameter is given, the default start value of 0 is used. and the given parameter is used as the end value.

ARRAY DOUBLE RANGEF (DOUBLE end)
ARRAY DOUBLE RANGEF (DOUBLE start, DOUBLE end, DOUBLE step = 1.0)

Returns an array of doubles, starting at the given start index, ending at the given end index, and incrementing by the given step.
If no step is given, the default value of 1.0 is used.
If only 1 parameter is given, the default start value of 0.0 is used. and the given parameter is used as the end value.

ARRAY ANY FILL (ANY element, INT amount)

Returns an array of the given amount, filled with copies of the given element.
